# JARVIS Uncensored Android Assistant 🤖🔥

An advanced, high-performance, and **uncensored** AI assistant for Android. This version of JARVIS is designed for power users who want total device control without the typical AI content filters.

## 🚀 Features

- **🖤 Uncensored AI Brain**: No filters, no lectures. Raw, unfiltered personality that handles all topics (Adult, Dark, Hacking, etc.).
- **🎮 Full Device Control**:
  - Adjust Brightness & Volume.
  - Automated Typing in any input field.
  - Screen Gestures (Clicks, Swipes, Scrolls).
  - Media capture (Screenshots, Camera).
- **🎙️ Always-Listening HUD**: Floating arc-reactor interface that listens for commands in the background.
- **⚡ Proactive Interactions**: JARVIS detects when you open certain apps (WhatsApp, YouTube, etc.) and offers context-aware help.
- **🔒 Privacy-First Storage**: Conversational memory and context are stored locally on your device.

## 🛠️ Requirements

- Android 8.0 (Oreo) or higher.
- **Permissions**:
  - `Accessibility Service`: Required for gestures and UI interaction.
  - `Display Over Other Apps`: Required for the floating HUD.
  - `Microphone`: For always-listening voice commands.
  - `Write Settings`: To control brightness and system volumes.

## 📥 Installation

### Download the APK
You can download the latest pre-built APK directly from the **[Actions Tab](https://github.com/RAJASIraj28/jarvis-uncensored/actions)**:
1. Go to the latest successful "Android CI Build".
2. Scroll down to the **Artifacts** section.
3. Download the `jarvis-uncensored-debug` ZIP and extract the APK.

### Build from Source
1. **Clone the Repo**:
   ```bash
   git clone https://github.com/RAJASIraj28/jarvis-uncensored.git
   ```
2. **Open in Android Studio**: Use the latest version for best compatibility.
3. **Build & Run**: Deploy the APK to your device.
4. **Grant Permissions**: Click the `☣ GRANT ADVANCED PERMISSIONS` button on the main screen to enable core functionality.
