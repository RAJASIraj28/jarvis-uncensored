# 🛡️ Security Policy

## Supported Versions

We actively provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| v3.5.x  | :white_check_mark: |
| < v3.5  | :x:                |

## Reporting a Vulnerability

As JARVIS handles advanced system permissions (Accessibility, Screen Recording, and Biometrics), security is our priority.

If you discover a potential security vulnerability, please do NOT open a public issue. Instead, report it privately via the following channels:

1.  **Email**: [Your Contact Email]
2.  **GitHub Private Report**: Use the "Report a vulnerability" button on the Security tab.

We will acknowledge your report within 48 hours and provide a timeline for a fix.

## Privacy Note
JARVIS stores all conversational memory and screen analysis data **locally** on your device. The only external communication is with the **OpenRouter API** for LLM processing. No personal data is harvested or sold.
